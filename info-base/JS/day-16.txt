.cosest() ищет ближайший элемент с заданным значением,
двигаясь по родителям. НЕ может искать по элементам
внутри одного родителя;

    можно задать для родительского тега с addEventListener на клик,
    чтобы кликая на любой дочерний элемент этого тега, клик засчитывался
    в первоначальный тег;

если нужно найти что-то внутри одного родителя можно использовать
либо previousElementSibling / nextElementSibling, либо
можно сперва подняться к родителю через parentElement, а затем
искать в нём нужное с помощью querySelector;
