data- можно писать в HTML тегах, как например в кнопках. помогает брать необходимый
контекст для кнопки, как в случае, когда кнопка является кнопкой покупки, а цена
написана выше: можно написать в теге кнопки data-price="1000", а затем взять это
значение в js с помощью event.target.dataset.price;


для поиска в js чего-то в HTML лучше использовать:

    event.target.parentElement.querySelector('.price')
