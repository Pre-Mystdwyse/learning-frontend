если не указать цвет box-shadow, то он возьмёт цвет из color;
это работает даже на ::after и ::before, ибо они наслудуют цвет;
даже если цвет был указан в теге, то он может смениться, если будет
указан иной в классе или id;


медиазапросы - это специальные правила, которые включаются только при
определённых условиях (например, если экран уже 600 пикселей);

    синтаксис:
        @media (max-width: 600px) {
            /*эти стили сработают только на экранах уже 600px */
            body {
                padding-top: 100px;
            }
        }
    
    можно менять значения переменных глобально для мобилок:
        @media (max-width: 600px) {
            :root {
                --header-height: 100px;
            }
        }

    overflow-x: auto; даёт возможность скроллить страницу
    вбок, не растягивая весь сайт;
