можно менять размер background, в т.ч. если он градиент;
монжо сместить один из цветов за блок, а потом при том же
наведении изменить размер или позицию, отчего получится
будто происходит некая заливка. пример реализации:

    <div class="test"></div>

    .test {
        background: linear-gradient(90deg, #962b82 0%, #green 50%, #c43ace 100%);
        background-size: 150%;
    }

    .test:hover {
        background-position: 40%;
    }
